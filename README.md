### Install
```
npm install v-easy-components -D
```
or
```
yarn add v-easy-components -D
```

### Quick Start
``` javascript
import 'v-easy-components/lib/theme-chalk/index.css'
import VEasy from 'v-easy-components'

Vue.use(VEasy);
```
